Dog Adoption System

The Dog Adoption System is a project we developed for our Data Structures and Algorithms course using C++. It is designed to assist both dog shelters and individuals looking to adopt a dog. The system allows users to track dogs currently available for adoption, as well as those that have already been adopted. It also categorizes dogs based on breed and age, making the adoption process more organized and efficient.

NU Finds

NU Finds is an Android application we developed during our Application Development course using Java, PHP, and SQL. The app is designed to support the students and SDO officers of NU Lipa in fostering a responsible and connected campus community. It enables users to report and track lost items by uploading item details to the system. If an item has already been turned in to the Student Discipline Office (SDO), it becomes searchable within the app. The system is backed by a structured database that stores essential information such as lost item details, student ID, full name, item description, and the location where it was found.
